Features

AI Avatar comes with a toolkit of autonomous agents that can perform various operations, completely autonomous and driven by a Language Model (LLM) or other types of foundational models.

Search Agent

Utilizes search engines such as Google, DuckDuckGo, and Metaphor for automated web searches.

Browser Agent

Equipped with the capability to scrape, read website contents, and interact with web pages and web applications.

X Agent

An account manager for managing your X social media profile.

Email Agent

Configured to access our preferred email provider, with the function to read, analyze, and respond to emails.

Gmail Agent

A specialized email agent for the Google Mail provider.

Google Drive Agent

An agent with accessibility to your preferred Google Drive, capable of summarizing, reading, and creating documents. It can also perform basic document management like setting up sharing preferences.

API Agent

Specialized in consuming various APIs given it's provided with appropriate documentation describing the API it's connecting to.

Coding Agent

Designed to write code, from a simple webpage to a Python game. It can write code based on prompts and subsequent questions; the prompts don't need to be technical.

Scrum Agent

Facilitates the entire Scrum process, manages team members, DS, and creates project breakdowns to Epics > User Stories > Development Tasks. It also helps in project scoping estimation and writing tech documentation.

Github Agent

Can load a github repositry into its system and allow you to ask questions about the provided code repository
